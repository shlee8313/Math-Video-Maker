# YouTube Uploader Skill

## 개요

완성된 영상의 유튜브 업로드에 필요한 메타데이터를 생성합니다.

## 입력

- `1_script/approved_script.json` (대본 원문)
- `state.json` (프로젝트 설정)

## 출력

1. 제목 (3가지 선택지)
2. 설명 (Description)
3. 태그 (Tags)
4. 썸네일 프롬프트
5. 업로드 설정

---

## 1. 제목 작성 규칙

### 3가지 스타일로 제안

| 스타일 | 특징 | 예시 |
|--------|------|------|
| **메인 (권장)** | 질문형 + 부제목 | "왜 X일까? \| Y의 수학" |
| **클릭 유도형** | 감정 자극, 호기심 | "X했더니 Y? 그 이유가 충격적입니다" |
| **교육형** | 키워드 중심, 검색 최적화 | "X의 비밀: Y가 알려주는 Z" |

### 제목 규칙

- 길이: 40~60자 (한글 기준)
- 핵심 키워드를 앞에 배치
- `|` 또는 `:` 로 부제목 구분
- 숫자가 있으면 포함 (클릭률 상승)
- 이모지 사용 금지 (검색 최적화에 불리)

---

## 2. 설명 (Description) 작성 규칙

### 구조

```
[Hook 문장 - 대본의 첫 문장 활용]

이 영상에서는 [주제]의 [핵심 내용]을 파헤칩니다.

📌 이 영상에서 배우는 것
• [핵심 포인트 1]
• [핵심 포인트 2]
• [핵심 포인트 3]
• [핵심 포인트 4]
• [핵심 포인트 5]

⏱️ 타임라인
0:00 - [섹션 1]
X:XX - [섹션 2]
...

🔔 구독과 알림 설정으로 [채널 특성] 영상을 받아보세요!

#태그1 #태그2 #태그3 ...
```

### 규칙

- 첫 2줄이 검색 결과에 노출됨 → 핵심 내용 배치
- 타임라인은 `scenes.json`의 섹션 구분 활용
- 해시태그는 설명 끝에 10~15개 배치
- 외부 링크는 필요시에만 추가

---

## 3. 태그 (Tags) 작성 규칙

### 태그 구성 (20~30개)

| 카테고리 | 개수 | 예시 |
|----------|------|------|
| **핵심 키워드** | 5~7개 | 주제 직접 관련 단어 |
| **영문 키워드** | 3~5개 | 영어 번역 버전 |
| **관련 개념** | 5~7개 | 연관 학문/분야 |
| **롱테일 키워드** | 5~7개 | 구체적인 검색어 |
| **채널 브랜딩** | 2~3개 | 채널명, 시리즈명 |

### 태그 규칙

- 쉼표로 구분
- 가장 중요한 키워드를 앞에 배치
- 너무 일반적인 태그 지양 (예: "영상", "유튜브")
- 대본의 `meta.key_equations`, `meta.wow_moments`에서 키워드 추출

---

## 4. 썸네일 프롬프트 작성 규칙

### Midjourney/DALL-E 프롬프트 구조

```
[메인 비주얼 설명], [감정/표정], [배경 요소], [수학 공식이나 숫자],
Style: [스타일 키워드], [색상], [구도], 16:9 aspect ratio, YouTube thumbnail style --ar 16:9 --v 6
```

### 필수 요소

| 요소 | 설명 |
|------|------|
| **메인 비주얼** | 주제를 한눈에 보여주는 이미지 |
| **감정 표현** | 놀람, 궁금함, 충격 등 (클릭 유도) |
| **대비 요소** | Before/After, 비교, 화살표 등 |
| **수학적 요소** | 핵심 공식이나 숫자 (선택) |
| **색상** | 고대비 (빨강+노랑, 파랑+주황 등) |

### 한글 텍스트 오버레이 제안

썸네일 프롬프트와 별도로, 포토샵에서 추가할 한글 텍스트 제안:
- **메인 텍스트**: 2~4글자, 임팩트 있는 단어
- **서브 텍스트**: 부연 설명 (선택)
- **숫자/가격**: 변화를 보여주는 수치

### 스타일 가이드 (프로젝트 설정 기반)

| 프로젝트 스타일 | 썸네일 스타일 키워드 |
|----------------|---------------------|
| minimal | clean, white background, simple shapes, minimalist |
| cyberpunk | neon glow, dark background, blue and pink, futuristic |
| paper | paper cutout, soft shadows, warm colors, craft style |
| space | cosmic, stars, deep blue, nebula, dramatic lighting |
| geometric | abstract shapes, bold colors, modern, flat design |

## 5. 업로드 설정

### 기본 설정

| 항목 | 권장값 |
|------|--------|
| 카테고리 | 교육 (Education) |
| 공개 범위 | 공개 |
| 아동용 콘텐츠 | 아니요 |
| 연령 제한 | 없음 |
| 유료 프로모션 | 아니요 |
| 댓글 | 허용 |
| 라이선스 | 표준 YouTube 라이선스 |

### 자막 설정

- 자동 생성 자막 활성화
- 한국어 기본 설정
- 영어 자막 추가 권장 (글로벌 시청자용)

---

## 실행 방법

### Claude에게 요청

```
"유튜브 업로드 정보 만들어줘"
또는
"youtube-uploader 실행"
```

### Claude가 할 일

1. `state.json`에서 프로젝트 정보 확인
2. `1_script/approved_script.json` 읽기
3. 위 규칙에 따라 다음 생성:
   - 제목 3가지
   - 설명 (타임라인 포함)
   - 태그 20~30개
   - 썸네일 프롬프트 2개
   - 한글 텍스트 오버레이 제안
   - 업로드 설정 체크리스트

## 참고

- 대본의 `meta` 섹션에서 `wow_moments`, `key_equations`, `main_metaphors` 활용
- 타임라인은 `scenes.json`의 섹션별 시작 시간 계산 필요
- 썸네일 스타일은 `state.json`의 `settings.style`과 일치시킬 것
